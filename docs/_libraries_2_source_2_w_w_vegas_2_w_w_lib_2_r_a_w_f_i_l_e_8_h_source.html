69/*
70** This is the definition of the raw file class. It is derived from the abstract base FileClass
71** and handles the interface to the low level DOS routines. This is the first class in the
72** chain of derived file classes that actually performs a useful function. With this class,
73** I/O is possible. More sophisticated features, such as packed files, CD-ROM support,
74** file caching, and XMS/EMS memory support, are handled by derived classes.
75**
76** Of particular importance is the need to override the error routine if more sophisticated
77** error handling is required. This is more than likely if greater functionality is derived
78** from this base class.
79*/

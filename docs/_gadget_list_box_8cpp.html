void GadgetListBoxGetSelected ( GameWindow * listbox,
Int * selectList )

Get the selected item(s) of a listbox. For a single select listbox the parameter should be a single integer pointer. For a multi select listbox the parameter should be an (Int *), this array returned will be an array of indices of the selected items of the list box. An entry of -1 in this array is the "end" of the selected list, and this list will never be larger than the max items in the list box

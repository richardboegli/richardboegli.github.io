41// Desc: Helpful interface for RadioButtons
42//
43// RadioButton IMAGE/COLOR organization
44// When control is enabled:
45// enabledDrawData[ 0 ] is the background image for the whole enabled control
46// enabledDrawData[ 1 ] is the enabled, unselected radio box
47// enabledDrawData[ 2 ] is the enabled, selected radio box
48//
49// When control is disabled:
50// disabledDrawData[ 0 ] is the background image for the whole disabled control
51// disabledDrawData[ 1 ] is the disabled, unselected radio box
52// disabledDrawData[ 2 ] si the disabled, selected radio box
53//
54// When control is hilited (mouse over it and enabled)
55// hiliteDrawData[ 0 ] is the background image for the whole hilited control
56// hiliteDrawData[ 1 ] is the hilited, unselected radio box
57// hiliteDrawData[ 2 ] is the hilited, selected radio box
58//
59// GadgetRadioGetEnabledImage // LEFT
60// GadgetRadioGetEnabledUncheckedBoxImage // Middle
61// GadgetRadioGetEnabledCheckedBoxImage // right
62//-----------------------------------------------------------------------------

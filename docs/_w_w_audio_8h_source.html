325 //
326 // Note: 3D sound effects should be inserted into the SoundScene.
327 // They should not be manually played/stopped/etc.
328 //
329 // True 3D sound effects require mono, uncompressed, WAV data.
330 // If the supplied data is not in this format the sound will
331 // become a pseudo-3D sound.
332 //
333 // Pseudo-3D sounds are not true 3D sounds. They have the
334 // same properties as 3D 'sound effects' but do not use 3D
335 // hardware, are not restricted to mono, uncompressed, WAV data,
336 // and do not calculate doppler and reverb effects.
337 //

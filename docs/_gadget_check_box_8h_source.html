41// Desc: Helpful interface for checkboxes
42//
43// CheckBox IMAGE/COLOR organization
44// When control is enabled:
45// enabledDrawData[ 0 ] is the background image for the whole enabled control
46// enabledDrawData[ 1 ] is the enabled, unselected check box
47// enabledDrawData[ 2 ] is the enabled, selected check box
48//
49// When control is disabled:
50// disabledDrawData[ 0 ] is the background image for the whole disabled control
51// disabledDrawData[ 1 ] is the disabled, unselected check box
52// disabledDrawData[ 2 ] si the disabled, selected check box
53//
54// When control is hilited (mouse over it and enabled)
55// hiliteDrawData[ 0 ] is the background image for the whole hilited control
56// hiliteDrawData[ 1 ] is the hilited, unselected check box
57// hiliteDrawData[ 2 ] is the hilited, selected check box
58//

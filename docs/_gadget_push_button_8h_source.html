41// Desc: Helpful interface for PushButtons
42//
43// PushButton IMAGE/COLOR organization
44// When control is enabled:
45// enabledDrawData[ 0 ] is the background image for the whole enabled control
46// enabledDrawData[ 1 ] is the enabled, selected button graphic
47//
48// When control is disabled:
49// disabledDrawData[ 0 ] is the background image for the whole disabled control
50// disabledDrawData[ 1 ] is the disabled, but selected button graphic
51//
52// When control is hilited (mouse over it and enabled)
53// hiliteDrawData[ 0 ] is the background image for the whole hilited control
54// hiliteDrawData[ 1 ] is the hilited, selected button graphic
55//
56//-----------------------------------------------------------------------------

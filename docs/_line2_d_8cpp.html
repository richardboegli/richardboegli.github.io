PointInsideRect3D will return true iff inputPoint lies iside of the rectangle specified by bl, tl, br, tr. It does not actually consider the Z value, it is merely a convenience function for calling PointInsideRect2D This function will take the ptToTest and will perform even-odd checking against the area. If the area is not closed, it will be closed for this check.

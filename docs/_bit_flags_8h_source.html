43/*
44 BitFlags is a wrapper class that exists primarily because of a flaw in std::bitset<>.
45 Although quite useful, it has horribly non-useful constructor, which (1) don't let
46 us initialize stuff in useful ways, and (2) provide a default constructor that implicitly
47 converts ints into bitsets in a "wrong" way (ie, it treats the int as a mask, not an index).
48 So we wrap to correct this, but leave the bitset "exposed" so that we can use all the non-ctor
49 functions on it directly (since it doesn't overload operator= to do the "wrong" thing, strangley enough)
50*/

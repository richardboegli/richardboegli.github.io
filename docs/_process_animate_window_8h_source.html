42// have to do is create a new class derived from ProcessAnimateWindow.
43// Then setup each of the virtual classes to process an AnimateWindow
44// class. The Update adn reverse functions get called every frame
45// by the shell and will continue to process the AdminWin until the
46// isFinished flag on the adminWin is set to true.
